# Changelog

All notable changes to the HeyGen MCP VS Code Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial VS Code extension release
- Automatic MCP server registration with VS Code
- HeyGen API key configuration command
- Help command for quick access to documentation
- Support for Python 3.10-3.13
- Integration with uv package manager for server execution

### Features
- **User Management**: Check account credits and subscription status
- **Voice Library**: List and discover available voices for video generation
- **Avatar Discovery**: Browse and filter avatars by type and style
- **Video Generation**: Create AI videos with custom avatars, voices, and backgrounds
- **Template Management**: List and use video templates
- **Asset Management**: Upload, list, and delete assets
- **Folder Organization**: Create, list, and manage asset folders

### Configuration
- `heygen-mcp.apiKey`: Store your HeyGen API key securely
- `heygen-mcp.autoStartServer`: Control automatic server startup (default: true)
